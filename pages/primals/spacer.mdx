import { Spacer, Text } from "@/primals"

# Spacer

The `Spacer` component is a simple component that adds space between elements. It is useful for adding space between elements in a layout.

```jsx
<Spacer p="10" />
```

## Props

This takes all the props from the default props table.

### Examples

In the below examples you can see three text items with a spacer between them.

<Spacer />
<Text>Item 1</Text>
<Spacer />
<Text>Item 2</Text>
<Spacer p="8" />
<Text>Item 3</Text>
